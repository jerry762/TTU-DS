## Introduction:

資料結構是電腦中儲存資料與組織資料的方式,每一種資料結構都有其特性,而我們必須了解其特性,才能在以後的軟體開發中選對容器,一旦選錯,則會大大的影響程式的效率與維護,因此學好資料結構對於一個合格的軟體工程師來說是非常重要的。


- Data Structure
    - Primitive Data Structure
        - Integer
        - Float
        - Character
        - Boolean
    - Non-Primitive Data Structure
        - Linear
            - Static
                - Array
            - Dynamic
                - Linked list
                - Queue
                - Stack
        - Non Linear
            - Tree
            - Graph
            - Table
            - Set
